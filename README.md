# logica-programacion-1
<p>Crear un programa en Javascript que realice lo siguiente:
Debe solicitar al usuario 3 números por prompt y guardarlos en sus respectivas variables.
Debe analizar los números, identificar cual es el número mayor, el número del centro y el número menor.
Debe imprimir los números por consola o por el DOM ordenados de mayor a menor, y de menor a mayor.
Debe ser capaz de identificar si los números son iguales e imprimir un mensaje por consola o por el DOM diciendo que los números son iguales.
Prueba tu programa con las siguientes entradas para asegurarte que funcione correctamente:</p>
<ul>
  
  <li>Entradas: 
  4, 4, 2.
  <li>Salida:
  4, 4, 2.
  2, 4, 4.
  <li>Entradas: 
  4, 2, 4.
  <li>Salida:
  4, 4, 2.
  2, 4, 4.
  <li>Entradas: 
  2, 4, 4.
  <li>Salida:
  4, 4, 2.
  2, 4, 4.

</ul>
<p>Pruébalo con las combinaciones de números que se te ocurran.
Al final de tu práctica, tienes que subir el ejercicio a tu repositorio de GitHub.
Colócalo en un repositorio llamado “logica-programacion-1”</p>
